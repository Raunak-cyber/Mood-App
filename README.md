# Moodzy 💖

Moodzy is a web-based emotional support platform designed to help users manage mood fluctuations, especially during periods.

## 🌸 Features
- Mood-based suggestions
- Relaxation music
- Mini games
- Clean and aesthetic UI

## 🎯 Purpose
Moodzy helps users feel better by providing calming experiences like music, games, and mood tracking.

## 🛠 Tech Used
- HTML
- CSS
- JavaScript

## 🚀 Future Plans
- AI mood detection
- Personalized suggestions
- User login system with database
